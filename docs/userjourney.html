User Journey

User flow documentation for Season V2.

New User Flow

1 Visit /launch — See countdown, click "Join Waitlist"
2 Enter email — Sign up for waitlist
3 Receive invite — Click link in email (/invite/:token)
4 Complete onboarding — 11 steps collecting cycle info
5 Access app — Redirected to /calendar

Returning User Flow

1 Log in — /session/new
2 View calendar — See cycle phase, upcoming events
3 Log symptoms — Daily tracking

Key Screens

  • Calendar — Monthly/weekly view with phase colors
  • Tracking — Self analysis with donut wheel
  • Symptoms — Mood, physical, mental logging
  • Superpowers — Track positive symptoms
  • Streaks — Gamification
  • Settings — Profile, calendar, notifications
